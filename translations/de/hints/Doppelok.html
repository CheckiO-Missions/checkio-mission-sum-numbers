Hier ist eine Lösung mit der Funktion sum() und der Verwendung der Funktionen map() und filter() anstelle des Listengenerators. In der Funktion filter() überprüfen wir mit der Methode str.isdigit(), ob das String-Element eine Zahl ist. 

def sum_numbers(text: str) -> int:

    return sum(map(int, filter(str.isdigit, text.split())))

Anstelle einer if Bedingung können wir auch einen try/except Ausdruck verwenden. Wenn die Konvertierung einer Zeichenkette in eine Zahl int(i) fehlschlägt, überspringen wir einfach diesen Schritt und fahren mit einem anderen Element der Zeichenkette fort. 

def sum_numbers(text: str) -> int:

    result = []
    for i in text.split():
        try:
            result.append(int(i))
        except:
            pass
    return sum(result)
