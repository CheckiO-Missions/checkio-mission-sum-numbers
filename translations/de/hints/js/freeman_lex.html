💡 Die Idee ist, den gegebenen Text durch Leerzeichen in Wörter aufzuteilen und jedes davon in eine Zahl umzuwandeln. Wenn die Umwandlung korrekt ist - fügen Sie diese Zahl zum Ergebnis hinzu.

Ok, womit soll ich anfangen?

Deklarieren Sie eine neue Variable res mit dem Anfangswert 0. Das ist Ihre Summe der richtigen Zahlen, die Ihre Funktion zurückgeben soll. 

let res = 0;
...
return res

Ok, was ist als nächstes zu tun?

Verwenden Sie die Methode split() mit dem Argument Leerzeichen, um den Text nach Wörtern aufzuteilen und for/of loop, um über diese Wörter als Werte der Variablen str zu iterieren. 

for (let str of text.split(" ")){...}

Ich brauche einen kleinen Hinweis.

Sie haben Wörter als String. Innerhalb der Schleife müssen Sie nun jedes dieser Wörter mit der Methode Number() in eine Zahl umwandeln und einer neuen Variablen num zuweisen. 

for (let str of text.split(" ")){
    let num = Number(str);
    ...
}

Wie können Sie überprüfen, ob die Umwandlung korrekt ist?

Hier gibt es eine etwas knifflige Logik. Gehen wir vom Gegenteil aus. num muss eine Zahl sein, wir können also sagen, dass es NICHT NaN (keine Zahl) sein kann. Für eine solche Überprüfung benötigen Sie die Methode isNaN() (die true zurückgibt, wenn der Wert keine Zahl ist) und den Operator ! (not), um das Ergebnis der Methode zu invertieren. 

if (!isNaN(num))
    ...

Wenn die Prüfung true lautet - fügen Sie die Zahl num zum Ergebnis res hinzu. 

res += num

Wie sollte also meine endgültige Lösung aussehen?

Sie müssen nur Ihren Code in einen Funktionsausdruck verpacken und alle vorherigen Hinweise verbinden.

Hier ist die gesamte Lösung:

function sumNumbers(text: string): number {
    
    let res = 0;
    for (let str of text.split(" ")){
        let num = Number(str);
        if (!isNaN(num))
            res += num
    }   
    return res
}

Klar, schnappen Sie es sich!

Erstellen Sie aus dem anfänglichen Text ein Array, das nur aus Ziffernblöcken besteht, indem Sie einen regulären Ausdruck verwenden. Beginnen Sie dann mit 0, wandeln Sie jeden Block mit + in eine Zahl um und ermitteln Sie die Summe. 

function sumNumbers(t: string): number {

    return (t.match(/\b\d+\b/g)||[]).reduce((a,c)=>a+ +c,0);
}

Diese Lösung kann in einer Zeile geschrieben werden, aber um sie eher schmal als breit zu machen, wird jede Methode in einer neuen Zeile geschrieben. Teilen Sie den Text auf, filtern Sie nur gültige Zahlen und reduzieren Sie sie mit der Summenfunktion, die jeden String in eine Zahl umwandelt. Die anonyme Funktion wird auch hier verwendet. 

var sumNumbers = (test: string): number => test
    .split(' ')
    .filter(Number)
    .reduce((a, i) => a + Number(i), 0);
