你所要做的就是将代码包裹在一个函数表达式中，并返回结果。你可以简单地将我们的result 变量添加到return 表达式中的sum() 函数中。 

def sum_numbers(text: str) -> int:

    result = []
    for i in text.split():
        if i.isdigit():
            result.append(int(i))
    return sum(result)

下面是一个使用sum() 函数的解决方案，并使用map() 和filter() 函数来代替列表生成器。在filter() 函数中，使用str.isdigit() 方法，我们检查字符串元素是否是一个数字。 

def sum_numbers(text: str) -> int:

    return sum(map(int, filter(str.isdigit, text.split())))

我们可以使用一个if 条件，而不是一个try/except 表达式。如果将一个字符串转换为数字int(i) ，就会失败，我们只需跳过这一步，继续处理字符串的另一个元素。 

def sum_numbers(text: str) -> int:

    result = []
    for i in text.split():
        try:
            result.append(int(i))
        except:
            pass
    return sum(result)

或者你可以使用一个int 变量立即修改代码，不使用列表。
