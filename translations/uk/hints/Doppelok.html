Ось рішення з використанням функції sum() та замість генератора списку використовують функцію map() та filter(). У функції filter(), за допомогою методу str.isdigit() перевіряємо на те чи елемент рядку число. 

def sum_numbers(text: str) -> int:

    return sum(map(int, filter(str.isdigit, text.split())))

Замість умови if ми можемо використати вираз try/except. Якщо конвертувати рядок у число int(i) не вдається, то ми просто пропускаємо цей крок та переходимо до іншого елементу рядку. 

def sum_numbers(text: str) -> int:

    result = []
    for i in text.split():
        try:
            result.append(int(i))
        except:
            pass
    return sum(result)

Або ж можна модифікувати код одразу з використанням змінної типу int, без списку. 

def sum_numbers(text: str) -> int:

    result = 0
    for i in text.split():
        try:
            result += int(i)
        except:
            pass
    return result
