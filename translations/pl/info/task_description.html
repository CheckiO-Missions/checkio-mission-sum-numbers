Twoim zadaniem jest wyznaczenie sumy liczb występujących w tekście. Weź pod uwagę tylko oddzielone liczby - liczby połączone ze słowami nie powinny być sumowane.

Tekst składa się z liczb, spacji i słów w języku angielskim.

Wejście: Ciąg znaków (string).

Wyjście: Liczba całkowita (integer).

Przykład:

{% if interpreter.slug == "js-node" %} 
sumNumbers('hi') == 0
sumNumbers('who is 1st here') == 0
sumNumbers('my numbers is 2') == 2
sumNumbers('This picture is an oil on canvas '
 'painting by Danish artist Anna '
 'Petersen between 1845 and 1910 year') == 3755
sumNumbers('5 plus 6 is') == 11
sumNumbers('') == 0
{% else %} 
sum_numbers('hi') == 0
sum_numbers('who is 1st here') == 0
sum_numbers('my numbers is 2') == 2
sum_numbers('This picture is an oil on canvas '
 'painting by Danish artist Anna '
 'Petersen between 1845 and 1910 year') == 3755
sum_numbers('5 plus 6 is') == 11
sum_numbers('') == 0
{% endif %}
