Nie wiem, jak zacząć rozwiązywać tę misję.

Zadanie polega na znalezieniu w wierszu elementów zawierających tylko liczby, bez liter, i zwróceniu sumy tych liczb.

Potrzebuję pomocy, aby kontynuować misję.

Aby rozwiązać misję, użyjemy generatora list. Przydatne będzie zapoznanie się z informacjami na temat: list comprehension (example), str.split() (example), str.isdigit() (example), int(), (example).

Zapiszmy więc, co dokładnie powinno znaleźć się w naszym wyrażeniu generatora listy. W pętli for będziemy iterować po łańcuchu podzielonym na listę przy użyciu metody str.split(). 

result = []  # This is where our result of the cycle will be stored
for i in text.split():
    ...

Nie wiem już, co robić. Potrzebuję małej podpowiedzi.

Musi Pan dodać sprawdzanie, czy element string jest liczbą, czy nie. 

result = []
for i in text.split():
    if i.isdigit():
        ...

Nic nie działa. SOS

Tak więc, jeśli sprawdzenie się powiedzie, dodajemy element string do listy, po uprzednim przekonwertowaniu go na typ danych "integer" za pomocą funkcji int(). 

result = []
for i in text.split():
    if i.isdigit():
        result.append(int(i))

Nie chcę się poddawać, proszę o pomoc!

Wszystko, co musi Pan zrobić, to zawinąć kod w wyrażenie funkcji i zwrócić wynik. Może Pan po prostu dodać naszą zmienną result do funkcji sum() w wyrażeniu return. 

def sum_numbers(text: str) -> int:

    result = []
    for i in text.split():
        if i.isdigit():
            result.append(int(i))
    return sum(result)

Można też odejść od zmiennej i zawinąć ją w zrozumienie listy. 

def sum_numbers(text: str) -> int:

    return sum(int(i) for i in text.split() if i.isdigit())

Dobra, dobra! That's the spirit!=)
Oto rozwiązanie wykorzystujące funkcję sum() i używające funkcji map() i filter() zamiast generatora listy. W funkcji filter() za pomocą metody str.isdigit() sprawdzamy, czy element ciągu jest liczbą. 

def sum_numbers(text: str) -> int:

    return sum(map(int, filter(str.isdigit, text.split())))

Chcę więcej!

Zamiast warunku if możemy użyć wyrażenia try/except. Jeśli konwersja ciągu na liczbę int(i) nie powiedzie się, po prostu pomijamy ten krok i przechodzimy do kolejnego elementu ciągu. 

def sum_numbers(text: str) -> int:

    result = []
    for i in text.split():
        try:
            result.append(int(i))
        except:
            pass
    return sum(result)

Można też od razu zmodyfikować kod, używając zmiennej int, bez listy. 

def sum_numbers(text: str) -> int:

    result = 0
    for i in text.split():
        try:
            result += int(i)
        except:
            pass
    return result
