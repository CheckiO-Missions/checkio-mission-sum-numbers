All you have to do is wrap the code in a function expression and return the result. You can simply add our result variable to the sum() function in the return expression. 

def sum_numbers(text: str) -> int:

    result = []
    for i in text.split():
        if i.isdigit():
            result.append(int(i))
    return sum(result)

Here is a solution using the sum() function and using the map() and filter() functions instead of the list generator. In the filter() function, using the str.isdigit() method, we check whether the string element is a number. 

def sum_numbers(text: str) -> int:

    return sum(map(int, filter(str.isdigit, text.split())))

Instead of an if condition, we can use a try/except expression. If converting a string to a number int(i) fails, we simply skip this step and move on to another element of the string. 

def sum_numbers(text: str) -> int:

    result = []
    for i in text.split():
        try:
            result.append(int(i))
        except:
            pass
    return sum(result)
