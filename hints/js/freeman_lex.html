I need a small hint.

You have words as string. Now inside loop you need to convert each of them into number with Number() method and assign to a new variable num. 

for (let str of text.split(" ")){
    let num = Number(str);
    ...
}

How to check, if the conversion is correct?

Here is a little tricky logic. Let's think from the opposite. num have to be a number, so we may say, it can NOT be NaN (not a number). To make such checking, you need isNaN() method (return true if the value is not a number) and ! (not) operator to invert method result. 

if (!isNaN(num))
    ...

If the checking is true - add the number num to the result res. 

res += num

So, how my final solution should look like?

You just need to wrap your code into function expression and join all previous hints.

Here is the whole solution:

function sumNumbers(text: string): number {
    
    let res = 0;
    for (let str of text.split(" ")){
        let num = Number(str);
        if (!isNaN(num))
            res += num
    }   
    return res
}

Create an array from the initial text with digit blocks only using regular expression, then start from 0, convert each block into number with + and find the sum. 

function sumNumbers(t: string): number {

    return (t.match(/\b\d+\b/g)||[]).reduce((a,c)=>a+ +c,0);
}

This solution may be written in one line, but to make it more narrow then wide, every method is written from the new line. Split the text, filter only valid numbers and reduce them with the sum function converting each string into number. The anonymous function is also used here. 

var sumNumbers = (test: string): number => test
    .split(' ')
    .filter(Number)
    .reduce((a, i) => a + Number(i), 0);
